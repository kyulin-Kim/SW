텍스트 스타일

color 속성: 텍스트의 색상을 지정할 수 있다.

색상값 참조 사이트: 사이트로 이동

  • 색상명으로 지정
  • 16진수 값으로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정

text-decoration 속성: 텍스트에 줄을 긋거나 없앨 때 사용한다.

  • 하이퍼링크 밑줄 없애기
  • 텍스트영역 아래 줄 긋기
  • 텍스트 영역 중간에 줄 긋기
  • 텍스트 영역 위에 줄 긋기

text-transform 속성: 텍스트에 그림자 효과를 줄 때 사용한다.
